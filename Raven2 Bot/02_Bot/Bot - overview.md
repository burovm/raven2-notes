## Project Goal

Raven2 Auction Bot — это инструмент для анализа и автоматизации торговли террамондами в игре Raven 2.  
Бот собирает историю цен с аукциона, находит выгодные моменты для покупки и продажи и может:  
- работать как ассистент (давать сигналы игроку);  
- или в авто-режиме имитировать действия игрока и сам совершать сделки.

## Key Features (planned)

### 1. Data Collection

- Периодический сбор данных с аукциона террамондов (через скриншоты/захват окна клиента).  
- Парсинг лотов:
  - тип террамонда (item_type);
  - цена (price);
  - количество (quantity);
  - время снятия данных (timestamp);
  - направление сделки (side: buy/sell);
  - дополнительные поля (seller, extra), если доступны.
- Сохранение данных в локальную БД SQLite (`data/raven2.db`, таблица `lots`).

### 2. Analytics & Strategy

- Расчёт статистики по каждому типу террамонда:
  - средние цены;
  - минимумы/максимумы;
  - изменение цен по дням недели и времени суток.
- Поиск паттернов:
  - периоды, когда цена обычно ниже (окна для закупки);
  - периоды, когда цена выше (окна для продажи).
- Настраиваемые стратегии:
  - фильтрация по типу террамонда;
  - минимальная маржа;
  - лимиты по объёму/риску.

### 3. Assistant Mode

- Генерация рекомендаций:
  - какие террамонды покупать сейчас, по какой цене и в каком объёме;
  - какие террамонды продавать сейчас и по какой цене.
- Вывод сигналов:
  - в лог/консоль;
  - в файлы;
  - (в будущем) в Telegram/Discord-бота для уведомлений.

### 4. Auto-Execution Mode

- Модуль автокликера, который:
  - открывает аукцион в клиенте;
  - находит нужный террамонд;
  - покупает/выставляет лоты по рассчитанным параметрам.
- Имитация человеческого поведения:
  - случайные задержки и вариации движений;
  - ограничение времени работы и частоты операций;
  - приоритет безопасного поведения, чтобы снизить риск бана.

### 5. Tooling & Observability

- Инспекция БД:
  - просмотр последних лотов;
  - простые отчёты по ценам и объёмам.
- Логи действий бота:
  - какие сигналы были сгенерированы;
  - какие сделки были совершены в авто-режиме.
- Быстрый переключатель:
  - режим только-аналитика/ассистент;
  - режим с автоматическим исполнением сигналов.
