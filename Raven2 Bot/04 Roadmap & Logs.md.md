# Roadmap & Logs

## Roadmap

### Milestone 1 — Data collector MVP
- [ ] Настроить SQLite БД и таблицу `lots`.
- [ ] Сделать простой модуль `data` для записи лотов в БД.
- [ ] Определиться со способом захвата аукционного окна (скриншоты / захват окна).
- [ ] Реализовать минимальный `ui_capture`, который:
  - берёт скрин аукциона,
  - извлекает хотя бы один тестовый лот (пусть даже вручную/полуручно),
  - записывает его в таблицу `lots`.

### Milestone 2 — Analysis
- [ ] Собрать достаточный объём данных по террамондам.
- [ ] Написать простые агрегаты: средняя цена по дням, по неделям.
- [ ] Реализовать базовую стратегию сигналов buy/sell.

### Milestone 3 — Executor
- [ ] Выбрать библиотеку для автокликера.
- [ ] Описать безопасный сценарий исполнения сделки.
- [ ] Реализовать исполнение сигналов на одном персонаже с логированием.
