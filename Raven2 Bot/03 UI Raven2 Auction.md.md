# UI Raven2 Auction — Notes

Цель: описать интерфейс аукциона Raven 2 для террамондов, чтобы понимать, как его парсить (через скриншоты/захват экрана).

## Что нужно зафиксировать

1. Основное окно аукциона:
   - Где располагается список лотов (область экрана).
   - Где показываются: название террамонда, цена, количество, продавец.
   - Какие есть вкладки/фильтры (например, по типу предмета, по цене, по сортировке).

2. Окно покупки/продажи:
   - Где вводится цена.
   - Где указывается количество.
   - Где кнопки подтверждения (Buy/Sell/OK).

3. Фильтры и поиск:
   - Есть ли строка поиска по названию террамонда.
   - Есть ли переключение «покупка/продажа» или отдельные вкладки.

## План по скринам

- Сделать несколько скринов:
  - общий вид аукциона с террамондами;
  - скрин со списком лотов при разной сортировке;
  - скрин окна покупки;
  - скрин окна продажи.

Для каждого скрина:
- Подписать, какие зоны важны для парсинга (словами, с координатами позже).
- Зафиксировать, какие данные можно достать (название, цена, количество, продавец и т.д.).
