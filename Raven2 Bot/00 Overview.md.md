# Raven2 Auction Bot — Overview

Цель проекта: создать бота для Raven 2, который анализирует статистику продаж террамондов по времени (дни/недели), находит выгодные моменты для покупки и продажи и может автоматически или полуавтоматически совершать сделки.

Текущий статус:
- Настроен сервер rocketsrv с 2×RTX 4090.
- Создан проект на сервере: `~/projects/raven2-bot`.
- Инициализирован Git-репозиторий и базовая структура папок.
- Настроен Obsidian, vault хранится в папке Яндекс Диска.

Milestones:
1. Data collector MVP — сбор данных с аукциона террамондов и запись в БД.
2. Аналитика цен — поиск недельных паттернов и генерация сигналов buy/sell.
3. Executor — безопасный автокликер, который по сигналам совершает сделки в игре.
4. Тюнинг стратегий и антибан-логика.
